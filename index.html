Справка

Добавить значение в список можно с помощью формы в левой части приложения и кнопки «Добавить». Чтобы добавить сразу несколько значений, следует вводить каждое новое значение на новой строке. Пустые значения и строки, начинающиеся с пробела добавлять нельзя.

После добавления значений они появятся в списке в правой части приложения. Любой из элементов списка можно удалить, нажав на иконку с баком. Для выбора случайного элемента списка нажмите на кнопку «Определить победителя». Победитель будет отображен в верхней части приложения.
